# Calculate the Rey AVLT Accuracy

Calculates accuracy from true positives and false positives, but applies
`valid_values_only()` first. As percentage, i.e. a value of 55 means
55%.

## Usage

``` r
calc_REYAREC(reytcor, reyfpos)
```

## Arguments

- reytcor:

  true positives

- reyfpos:

  false positives
