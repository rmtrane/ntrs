# ntrs: Neuropsychological Test Result Standardization

S7 based framework for working with and standardizing neuropsychological
test results. Built in tools for working with the NACC Uniform Data Set
(UDS) and the National Alzheimer's Coordinating Center (NACC)
neuropsychological battery, but with extensibility at the center of the
design.
