# Calculate Rey Total

A short description...

## Usage

``` r
calc_REYTOTAL(rey1rec, rey2rec, rey3rec, rey4rec, rey5rec)
```

## Arguments

- rey1rec:

  A `REY1REC` object, or a numeric vector.

- rey2rec:

  A `REY2REC` object, or a numeric vector.

- rey3rec:

  A `REY3REC` object, or a numeric vector.

- rey4rec:

  A `REY4REC` object, or a numeric vector.

- rey5rec:

  A `REY5REC` object, or a numeric vector.

## Value

A `REYTOTAL` object representing the sum of the provided `rey` records
after error codes have been removed.
