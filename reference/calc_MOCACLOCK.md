# Calculate MoCA clock drawing test total

Sum of MOCACLOC, MOCACLON, and MOCACLOH, but applies removes any values
not 0 or 1.

## Usage

``` r
calc_MOCACLOCK(MOCACLOC, MOCACLON, MOCACLOH)
```

## Arguments

- MOCACLOC:

  values for clock contour

- MOCACLON:

  values for clock numbers

- MOCACLOH:

  values for clock hands
