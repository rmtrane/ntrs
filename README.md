The goal of `ntrs` is to provide a streamlined framework for working with neuropsychological test results and standardization of these. It is build around the NACC UDS battery, but is designed to make extension relatively painless for interested parties. 

## Installation

You can install the development version of ntrs from [GitHub](https://github.com/rmtrane/ntrs) with:

``` r
# install.packages("pak")
pak::pak("rmtrane/ntrs")
```

## Example

This is a basic example which shows you how to solve a common problem:

``` r
library(ntrs)
## basic example code
```
